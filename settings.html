This extension needs to download currency rates to work. These data are automatically fetched from currencylayer.com webstite. In order to make these calls, you must create an account on it and paste the API access key in the field below. Without this key, we cannot download rates and make your conversions.

Creating an account on currencylayer.com is fast, simple and free. Just :
  • head on their signup page and fill-in the form.
  • visit the dashboard page
  • copy the API access key (32 alphanumeric characters), paste in the field below
  • click the Save button
